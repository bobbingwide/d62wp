== Description ==
Drupal 6 to WordPress migration support tool

You have a Drupal installation that you want to convert to WordPress
This tool HELPS you to do it.

* To investigate the contents of the Drupal site 
* To help you create a WordPress plugin that registers the required Taxonomies, Custom Post Types (CPT aka CCT) and Fields
* To import content into WordPress


== Installation ==
1. Upload the contents of the d62wp plugin to the `/wp-content/plugins/d62wp' directory
1. Activate the d62wp plugin through the 'Plugins' menu in WordPress
